About Tartarus

Tartarus is a self-hosted, Memrise-style vocabulary trainer, written by Bahman Farhadian. It keeps everything — your word lists, scores, and practice history — in a single local SQLite database that you fully own and control.

Unlike Memrise, there's no account, no cloud, and no pre-recorded audio to wait for: every word is spoken on the fly using macOS say, in the right language automatically. Just write your own word list as a simple JSON file and start practicing.

How practice works

Every word has a score from 1.0 (struggling) to 9.0 (mastered). The score decides what kind of question you get — Learning, Audio, or Production — so a single session naturally mixes new words with ones you're reviewing. Get it right and the score goes up (and the question gets harder); get it wrong and it drops back down. See the tab for your history, or read the full scoring breakdown in the project's README.

The database is the source of truth

The CLI (tartarus.sh) and this web UI (tartarus_web.sh) both read and write the exact same data/tartarus.db SQLite file. Every answer updates the database immediately, and ending a session — whether by clicking !! here, pressing Ctrl+C in the terminal, or letting a session finish naturally — writes the session summary to the database too. You can freely switch between the CLI and the web UI; nothing is ever lost.

Tracking your progress

The tab shows your session history — sessions, time spent, words practiced, and correct/incorrect/drilled counts, per day and in total. Add a language there too, and you'll also see that word list's current words: each one's score, growth gauge, and full practice history (times practiced, correct, incorrect, drilled, flagged, and mastered).

Multiple word lists per language

A single user can have as many word lists as they like. The Word list field on the Practice page is just a list name — you can use anything: german_home, german_health, english_b2. Each name gets its own JSON file and its own score history in the database, so you can practice each topic independently. If the list name doesn't auto-detect as a language for audio (e.g. german_home), fill in the optional Audio language field (e.g. german) to get the right voice.

Customizing your word lists

Use the tab to create a new list and edit its words and definitions directly — no JSON editing required. Each word can have up to two definitions (e.g. an English meaning and a German one), and German nouns written with their article (der/die/das) are color-coded by gender automatically.

Keyboard-friendly

Every page's main button doubles as the Enter key on its input fields — start a session, load a report, or create and load a word list without touching the mouse. If you press Enter while a required field is empty, focus jumps to that field instead of submitting.

This web UI

Runs entirely on 127.0.0.1:9999 — standard library Python only, no external dependencies, no data ever leaves your machine.
